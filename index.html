    • Bienvenido a EcoGardens

    Nuestra mision es ayudar a las personas que quieren un jardin y no cunetan con el epacio suficiete EcoGardens les brinda la ayuda para poder crear un jardin vertical y con auto regado.

    EcoGardens es una starup dedficada a la creacion de jardines verticales automatizados, ideales para personas que viven en apartamentos o casas con poco espacio exterior.
    El producto principal es un sitema modular que incluye riego automatico, sensores de humedad y una aplicacion movil para monitoriar el estado del jardin.

    Ejemplos de uso

    Caracteristicas

    El sistema de riego automatico que se ejecuta egun la necesidad de las plantas.

    Sensores integrados que monitorean la humedad del suelo y ajustan el riego en consecuencia

    El jardin vertical es modulas, lo que significa que se puede agregar a quitar secciones segun el espacio disponible.

    Una aplicacion movil que permite a los usuarios monitorear y controlar el sistema de riego, recibir notificaciones y consejos de cuidado.

    Disañado para ser energeticamente eficiente, utilizando energia minima para mantener las plantas saludables.

    Fabricado con materiales reciclados y saludables.

    El sistema es facil de instalar y viene von isntrucciones detalladas.

    Apto para una amplia variedad de plantas, desde hierbas culinarias hasta plantas ornamentales.
